# ShopSphere

Microservices-based E-Commerce Platform built with:

- ASP.NET Core 9
- MVC
- JWT Authentication
- OAuth 2.0
- OpenID Connect
- Entity Framework Core
- RabbitMQ
- gRPC
- YARP API Gateway
- Docker
- Kubernetes
- Azure

## Architecture

- API Gateway
- Auth Service
- Product Service
- Cart Service
- Order Service
- Notification Service
- MVC Frontend
